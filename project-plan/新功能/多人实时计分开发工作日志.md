## 📋 **项目概述**

### **项目目标**
将现有的单机掼蛋计分小程序升级为支持多人实时协作的版本

### **核心功能**
1. **房间系统** - 创建、加入、管理
2. **实时计分** - 多人同步计分  
3. **基础多人功能** - 玩家管理、座位分配
4. **历史记录** - 个人和房间历史

### **技术栈**
- **前端**：微信小程序原生开发
- **后端**：微信云开发（云函数 + 云数据库）
- **架构**：智能混合架构
