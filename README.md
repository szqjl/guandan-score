## 🎮 项目简介

这是一个专为**掼蛋游戏**设计的微信小程序计分工具，提供完整的游戏计分、历史记录管理和数据存储功能。

### 核心功能
- ✅ **游戏计分系统** - 支持过A制和把数制两种规则
- ✅ **历史记录管理** - 完整的游戏历史查询和统计
- ✅ **数据存储系统** - 本地数据持久化存储
- ✅ **用户界面系统** - 友好的横屏游戏界面

### 游戏规则
- **过A制**：从2到A的升级路径，A必打，有3次机会
- **把数制**：设置最大把数，支持"追不上"机制
- **计分按钮**：双上(+3)、T3(+2)、T4(+1)

```bash

# 创建新版本

# 版本回滚

```
